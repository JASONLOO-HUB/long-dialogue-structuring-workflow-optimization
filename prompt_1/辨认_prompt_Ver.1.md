# Steps

4. **标记：** 如果某个话题讨论了但没结论，请标记为 (无结论)。
5. **填充：** 把确认的内容填入大纲。

# Constraints
- 不要写被否决的方案。
- 保持大纲层级。
- 总结要详细。
- 不要写没有的内容。
